# Bruno Kostiuk

Professor do Ensino Superior | Pesquisador em Computação | Desenvolvedor de Software

Sou professor do Ensino Superior no Centro Universitário Internacional (UNINTER) e
pesquisador em Computação, com atuação nas áreas de Processamento de Linguagem Natural,
Aprendizado de Máquina e Engenharia de Software aplicada.

Atualmente sou Doutorando em Informática pela Pontifícia Universidade Católica do Paraná (PUCPR),
com pesquisa focada em sistemas de recomendação de testes de software baseados em
descrições textuais de falhas e testes.
